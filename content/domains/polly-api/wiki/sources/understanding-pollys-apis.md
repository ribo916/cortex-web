# Source: Understanding Polly's APIs

**File:** `raw/devhub-guides/understanding-pollys-apis.md`

## Summary

High-level overview of what Polly's API suite can do. Lists 6 core capability areas: customizable pricing scenarios, loan pricing and lock management, Polly UI iFrame, batch pricing, MI quotes, and real-time webhooks.

## Key claims

- APIs are designed around the full loan pricing workflow: configuration → pricing → locking → downstream integration
- Supports end-to-end automation when implemented with appropriate architecture
- iFrame can be launched in multiple modes (scenario, quoting, lock)
